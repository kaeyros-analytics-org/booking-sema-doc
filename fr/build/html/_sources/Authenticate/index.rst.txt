Authentification
=================

**Connexion utilisateur**

L'objectif ici est de permettre à tout utilisateur autorisé (réceptionniste, praticien, administrateur) d’accéder à la plateforme.

.. image:: ../images/image0FR.png



- En cas de mot de passe oublié un code est envoyé à l'adresse mail que l'utilisateur va renseigner, ce qui va lui permettre de récupérer l’accès à son compte:

.. image:: ../images/image1FR.png



.. image:: ../images/image2FR.png



- Une fois le code verifié, il crée un **nouveau mot de passe** pour finaliser la réinitialisation :

.. image:: ../images/image3FR.png
